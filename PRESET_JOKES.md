# PRESET_JOKES.md — запасные wav на топ-5 тем анекдотов

## Зачем

Если на демо отвалится LLM (Ollama) или RAG (ChromaDB), у нас должны быть **готовые wav-файлы** с самыми популярными темами анекдотов 1986 года. Просто проигрываем их с диска через AudioClient.PlayStream() — никакой генерации на лету.

Темы выбраны по критериям:
- Узнаваемость (любой русский человек знает персонажа)
- Гарантированно смешной панчлайн
- Короткий (до 15 сек, чтобы не утомлять)
- Тематика 1986 года (Штирлиц — как раз 1986 год выхода "17 мгновений весны" ретрансляцией, Вовочка — школа, новые русские — начало кооперации)

---

## Топ-5 тем и шаблон текста

### 1. Штирлиц
**Реплика-вступление Бурунова:**
> «Ну, значицца, Штирлиц... он вообще почти весь восемьдесят шестой по телевизору был, ну вы помните...»

**Сам анекдот (пример):**
> «Штирлиц подошёл к окну. Из окна дуло. Штирлиц закрыл окно. Дуло исчезло.»

**Жест:** `point` (указывает пальцем в окно, типа "вон, смотри")
**Панчлайн-жест:** резкий `fist` (закрыл окно)
**После:** squat от смеха

---

### 2. Вовочка
**Реплика-вступление:**
> «А вот Вовочка... ну, который из школы... этот, как его... хулиган...»

**Сам анекдот:**
> «Учительница спрашивает Вовочку: "Вовочка, какую оценку тебе поставить?" — "Пять, Марьиванна!" — "Ну, ты ещё посчитай." — "Ладно, четыре, но не меньше!"»

**Жест:** `relaxed` → на панчлайне `peace` (два пальца — "четыре")
**После:** лёгкое покачивание головой

---

### 3. Поручик Ржевский
**Реплика-вступление:**
> «Ну этот... поручик наш... Ржевский, да... он вообще к дамам постоянно лез...»

**Сам анекдот:**
> «Поручик Ржевский на балу подходит к Наташе Ростовой: "Наташа, а Наташа, давайте я вас поцелую!" — "Поручик, я же дама!" — "Ну, тем более!"»

**Жест:** `thumbs_up` перед репликой
**Панчлайн:** `peace` к Наташе

---

### 4. Новые русские
**Реплика-вступление:**
> «А это ещё тогда, в восемьдесят шестом, кооперативы пошли, ну, эти... в малиновых пиджаках...»

**Сам анекдот:**
> «Встречаются два новых русских. Один говорит: "Я вчера "Запорожец" купил!" — "Ну и как?" — "Крылья бабочки, малиновый цвет, всё дела!"»

**Жест:** `open_hand` (демонстрация "вот он, мой запорожец")
**Панчлайн:** `close_hand` (крылья сложил)

---

### 5. Чапаев
**Реплика-вступление:**
> «Ну, Чапаев, понятное дело... он вообще к нам из гражданской войны пришёл, но мы его любим...»

**Сам анекдот:**
> «Чапаев спрашивает Петьку: "Петька, рубль есть?" — "Нет, Василий Иваныч." — "А два есть?" — "Тоже нет." — "Ну вот, а ты говоришь — капиталистическая революция."»

**Жест:** `point` на Петьку
**Панчлайн:** `open_hand` (пустые ладони — денег нет)

## Как сгенерировать пресеты

После того как F5-TTS или Piper обучен на голос Бурунова, прогоним скрипт:

```bash
# На ноутбуке с GPU (или на G1 если CPU справляется)
python3 scripts/generate_presets.py \
  --topics shtirlits,volodka,rzhevsky,new_russians,chapaev \
  --output /home/unitree/burunov-bot/data/preset_wav/
```

Скрипт `scripts/generate_presets.py` должен:
1. Прочитать тексты из `presets/jokes.yaml` (каждая тема — intro + joke + жесты)
2. Для каждого текста дёрнуть `POST http://localhost:8001/synthesize_wav`
3. Сохранить как `{topic}_intro.wav` и `{topic}_joke.wav`
4. Дополнительно сохранить `{topic}.json` с метаданными: где панчлайн, какие жесты

## Как проиграть пресеты на демо

В `robot_controller.py` (или в новом `preset_player.py`):

```python
import wave
from unitree_sdk2py.go2.audio.audio_client import AudioClient

def play_preset(topic: str, audio_client: AudioClient):
    """Проиграть готовый пресет анекдота с диска."""
    path = f"/home/unitree/burunov-bot/data/preset_wav/{topic}_joke.wav"
    with wave.open(path, "rb") as wf:
        sr = wf.getframerate()
        pcm = wf.readframes(wf.getnframes())
    # G1 expects 16kHz mono 16-bit — наши wav уже в этом формате
    audio_client.PlayStream("burunov_bot", stream_id=0, pcm_data=pcm)
```

Fallback в `api.py` (RAG endpoint):

```python
@app.post("/tell")
def tell(topic: str):
    try:
        # Пытаемся через RAG
        return rag_pipeline.generate(topic)
    except Exception:
        # Если упало — отдаём пресет
        return {
            "text": "(пресет) " + load_preset_text(topic),
            "preset": True,
            "audio_path": f"/data/preset_wav/{topic}_joke.wav"
        }
```
