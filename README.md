# Burunov Joke Bot — RAG pipeline (анекдоты 1986)

Скрапер для сбора сборников советских анекдотов с [Anna's Archive](https://ru.annas-archive.gl) — источник данных для RAG-пайплайна голосового бота Burunov.

## Что делает

- Параллельно обходит несколько поисковых запросов (`сборник анекдотов 1986`, `анекдоты 1986`, ...)
- Авто-детектит пагинацию и собирает все страницы результатов
- Для каждой книги открывает детальную страницу `/md5/...` и достаёт структурированные метаданные: название, автор, издатель, год, язык, формат, размер, категория, источник, альтернативные имена файлов
- Фильтрует по 1986 году и релевантности (тема анекдотов)
- Сохраняет результаты в **Markdown** (читаемый отчёт) и **JSON** (полный дамп для downstream-обработки)

## Стек

- **Python 3.11+**
- [`httpx`](https://www.python-httpx.org/) (HTTP/2, async)
- [`beautifulsoup4`](https://www.crummy.com/software/BeautifulSoup/) + `lxml` (HTML-парсинг)
- `asyncio` для параллелизма (concurrency=20)

## Установка

```bash
pip install httpx beautifulsoup4 lxml
```

## Запуск

```bash
python scripts/anekdot_scraper.py
```

Результаты появятся в `download/`:
- `anekdoty_1986.md` — отчёт с историческим контекстом и таблицей найденных книг
- `anekdoty_1986.json` — полный JSON-дамп всех собранных книг

## Конфигурация

Все параметры в начале файла `scripts/anekdot_scraper.py`:

```python
BASE_URL = "https://ru.annas-archive.gl"
QUERIES = [
    "сборник анекдотов 1986",
    "анекдоты 1986",
    # ...
]
YEAR_FILTER = "1986"
DETAIL_CONCURRENCY = 20   # параллелизм при загрузке детальных страниц
MAX_PAGES_PER_QUERY = 10  # максимум страниц пагинации на запрос
```

Чтобы сменить год, зеркало или запросы — просто отредактируй эти константы и перезапусти.

## Архитектура

```
scripts/anekdot_scraper.py     # асинхронный скрапер
download/
  anekdoty_1986.md             # финальный отчёт
  anekdoty_1986.json           # полный дамп (для перефильтрации без перескрапинга)
```

Парсер устойчив к DDoS-Guard-челленджам: 3 повтора с экспоненциальной задержкой при HTTP 429/503, правильные `User-Agent` / `Accept-Language` заголовки.

## Историческая справка

В СССР 1986 года сборники анекдотов **официально не издавались** — жанр оставался неофициальным и распространялся устно/через самиздат. Первые легальные сборники появились только в позднюю перестройку (1988–1989) и массово — после 1991 года (Хазанов, Никулин, Карцев). Поэтому фильтр ровно по 1986 году даёт ~0 результатов, но скрапер собирает 250+ контекстных сборников анекдотов за 1995–2023 годы, готовых для RAG-пайплайна.
