# Обоснование бюджета — Burunov Bot

Медиа-проект Олега Сироты: робот Unitree G1 EDU, найденный в сарае
в 1986 году, рассказывает анекдоты эпохи голосом Сергея Бурунова.

## Что нужно оплатить

### 1. GPU-сервер для обучения TTS (основная статья)

**Зачем:** клонирование голоса Сергея Бурунова через fine-tune GPT-SoVITS
и/или обучение Piper VITS. Требует GPU, на CPU нереально.

**Что арендуем:**
- RunPod RTX 4090 (24 ГБ VRAM) или аналог
- Время: 8-12 GPU-часов
- Цена: $0.6/час на RunPod
- **Итого: $5-10**

**Альтернативы (если нужна оплата по юрлицу РФ):**
- Selectel GPU A100: $1.5/час × 10ч = $15
- Сбер Cloud V100: $1.5/час × 10ч = $15
- Облако Mail.ru V100: $1.2/час × 10ч = $12

### 2. Запас на демо (опционально)

Если основной режим — GPT-SoVITS на сервере (не edge):
- RunPod 4090 на время демо: 8 часов = $5
- На случай если WiFi упадёт — есть edge-fallback (Piper на G1)

**Итого бюджет: $10-20**

## Что делаем на GPU

| Этап | GPU-часы | Зачем |
|---|---|---|
| Whisper medium транскрибация | 0.2 ч | Перевод аудио Бурунова в текст |
| Demucs выделение вокала | 0.2 ч | Убрать фоновую музыку из исходников |
| Fine-tune GPT-SoVITS | 0.5-1 ч | Клон голоса Бурунова |
| Обучение Piper VITS | 2-3 ч | Edge-fallback модель для G1 |
| Тесты инференса | 0.5 ч | Проверка качества |
| Запас | 4-5 ч | На случай пересборок |
| **Итого** | **8-10 ч** | |

## Что НЕ нужно оплачивать

| Статья | Почему бесплатно |
|---|---|
| **RAG (Gemma + ChromaDB)** | Крутится на самом G1 (у него mini-PC 16+ ГБ RAM) |
| **Piper инференс** | Крутится на G1, CPU-only |
| **AudioClient.PlayStream** | Через unitree_sdk2, не требует доп.ресурсов |
| **Деплой кода на G1** | По SSH, бесплатно |
| **Анекдоты (датасет)** | Друг-скраппер собрал, MIT-лицензия |

## Архитектура: что где крутится

```
[GPU-сервер аренда 8-10ч]              [Unitree G1 — постоянно]
  - Обучение GPT-SoVITS                 - Ollama (Gemma 3 4B Q4)
  - Обучение Piper                      - ChromaDB (27k анекдотов)
  - Выход: burunov.ckpt + .onnx         - Piper ONNX (edge-fallback)
                                          - robot_controller.py
                                          - AudioClient → динамик Stanley
                                          - LocoClient → жесты
                                          - HandClient → кисти RH56DFTP
```

## Сравнение с альтернативами (если бы арендовали больше)

| Подход | Цена | Качество голоса | Надёжность демо |
|---|---|---|---|
| **Наш: GPT-SoVITS + Piper edge** | $10-20 | 9/10 | 🔥 Высокая |
| Только Piper edge | $3-5 | 7/10 | 🔥 Максимальная |
| Только GPT-SoVITS сервер | $10 | 9/10 | 🟡 Зависит от WiFi |
| Готовые облака (ElevenLabs) | $20+/мес | 9/10 | ❌ Зависит от API |

Наш вариант даёт **лучшее качество при минимальной цене** — потому что
основная нагрузка (RAG + edge TTS) крутится на самом роботе, а GPU
нужен только разово для обучения.

## Риски и страховки

1. **WiFi ляжет на демо** → переключаемся на Piper edge (на G1)
2. **GPT-SoVITS обрывается** → тот же Piper edge
3. **GPU-сервер недоступен в момент обучения** → есть 8-10 ч запаса
4. **Клон Бурунова не получился** → fallback на встроенный TTS G1
   (TtsMaker, но только CN/EN — не подходит для анекдотов)
