# Обоснование бюджета — Burunov Bot

Медиа-проект Олега Сироты: робот Unitree G1 EDU, найденный в сарае
в 1986 году, рассказывает анекдоты эпохи голосом Сергея Бурунова.

## Что нужно оплатить

### 1. GPU-сервер для обучения TTS (основная статья)

**Зачем:** клонирование голоса Сергея Бурунова через fine-tune GPT-SoVITS
и/или обучение Piper VITS. Требует GPU, на CPU нереально.

**Рекомендуемая конфигурация (Selectel):**
- GPU: **A100 40GB** (одна карта, пулы ru-7a / ru-9a)
- CPU: 16 vCPU
- RAM: 87 ГБ
- Диск: 768 ГБ Локальный SSD NVMe (в комплекте)
- ОС: Ubuntu 24.04 LTS GPU Driver 580 (GPU-optimized образ)
- Сеть: 1 публичный IP + облачный роутер

**Время аренды:** 6-8 часов
- Whisper + Demucs: 15-30 мин
- GPT-SoVITS fine-tune: 20-30 мин
- Piper VITS обучение: 2-3 ч
- Тесты, отладка, конвертация: 1-2 ч
- Запас: 1-2 ч

## Что делаем на GPU

| Этап | GPU-часы | Зачем |
|---|---|---|
| Whisper medium транскрибация | 0.2 ч | Перевод аудио Бурунова в текст |
| Demucs выделение вокала | 0.2 ч | Убрать фоновую музыку из исходников |
| Fine-tune GPT-SoVITS (10 epochs) | 0.5 ч | Клон голоса Бурунова |
| Обучение Piper VITS (3000 epochs) | 2-3 ч | Edge-fallback модель для G1 |
| Тесты инференса | 0.5 ч | Проверка качества |
| Запас | 2-3 ч | На случай пересборок |
| **Итого** | **6-8 ч** | |

## Что НЕ нужно оплачивать

| Статья | Почему бесплатно |
|---|---|
| **RAG (Gemma + ChromaDB)** | Крутится на самом G1 (у него mini-PC 16+ ГБ RAM) |
| **Piper инференс** | Крутится на G1, CPU-only |
| **AudioClient.PlayStream** | Через unitree_sdk2, не требует доп.ресурсов |
| **Деплой кода на G1** | По SSH, бесплатно |
| **Анекдоты (датасет)** | Друг-скраппер собрал, MIT-лицензия |

## Архитектура: что где крутится

```
[GPU-сервер Selectel A100 40GB — 6-8 ч аренды]
  - Whisper medium транскрибация
  - Demucs отделение вокала
  - Fine-tune GPT-SoVITS → burunov.ckpt + burunov.pth
  - Обучение Piper VITS → burunov.onnx
  - Выход: models.zip (~400 МБ)

[Unitree G1 — постоянно, бесплатно]
  - Ollama (Gemma 3 4B Q4)
  - ChromaDB (27k анекдотов)
  - Piper ONNX (edge-fallback, оффлайн)
  - robot_controller.py
  - AudioClient.PlayStream → динамик Stanley
  - LocoClient → жесты
  - HandClient → кисти RH56DFTP
```

## Что получите за деньги

- 🎙️ Голос Сергея Бурунова на роботе (реальный клон, не синтетический)
- 📚 27 000 анекдотов 1986 года в RAG-базе
- 🤖 Полностью автономный робот (после обучения — без облака)
- 🎭 Жесты + LED-индикация в такт речи
- ✋ Кисти рук Inspire RH56DFTP делают жесты ("кулак", "большой палец вверх")
- 📱 HTTP-управление с телефона (ввод темы → анекдот)

## Риски и страховки

1. **WiFi ляжет на демо** → переключаемся на Piper edge (на G1)
2. **GPT-SoVITS обрывается** → тот же Piper edge
3. **GPU-сервер недоступен в момент обучения** → есть 6-8 ч запаса
4. **Клон Бурунова не получился** → fallback на встроенный TTS G1
   (TtsMaker, но только CN/EN — не подходит для анекдотов)
5. **Обучение прервалось** → не использовать "Прерываемый сервер",
   только фиксированный с авто-удалением по окончании
