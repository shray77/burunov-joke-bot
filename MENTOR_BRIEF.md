# Burunov Bot — стек и план для ментора

**Суть проекта:** Unitree G1 EDU рассказывает анекдоты 1986 года голосом
Сергея Бурунова. Медиа-проект Олега Сироты (Истринская сыроварня).

---

## Полный стек (что где крутится)

```
┌─────────────────────────────────────────────────────────────────┐
│  GPU-сервер (один раз, 3-4 часа)                                │
│                                                                 │
│  Задача: обучить клон голоса Бурунова                          │
│                                                                 │
│  Этапы:                                                         │
│  1. Whisper medium → транскрибация 15-30 мин аудио Бурунова    │
│  2. Demucs → отделение голоса от музыки                        │
│  3. GPT-SoVITS fine-tune → burunov.ckpt + burunov.pth (~300 МБ)│
│  4. Piper VITS train → burunov.onnx (~60 МБ)                   │
│                                                                 │
│  Выход: models.zip (~400 МБ) — копируем на робота и забываем   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Unitree G1 EDU (Main PC, постоянно, бесплатно)                 │
│                                                                 │
│  ПО:                                                            │

│  • ChromaDB                         — 27k анекдотов 1986        │
│  • Piper ONNX (CPU-only)            — edge TTS, ~60 МБ          │
│  • unitree_sdk2_python              — родной SDK робота         │
│  • robot_controller.py              — оркестратор              │

│  Железо G1 (используем):                                       │

│  • RGB-лента 256 цветов             — в такт речи               │

│  • 43 степени свободы               — LocoClient для поз        │
│                                                                 │
│  Полный оффлайн. WiFi/интернет НЕ НУЖЕН на демо.               │
└─────────────────────────────────────────────────────────────────┘
```

## Что нужно оплатить (и почему)

**Только GPU-сервер для обучения TTS.** Всё остальное бесплатно.

Без GPU клон голоса Бурунова сделать нельзя — fine-tune GPT-SoVITS и
обучение VITS требуют CUDA. На CPU это займёт недели.

После обучения (разово, 3-4 часа) модели копируются на робота — и
сервер больше не нужен. Демо работает полностью автономно.

## План действий (после решения по бюджету)

### Шаг 1. Найти аудио Бурунова (1-2 часа)
- Источники: интервью на YouTube, подкасты, закадровая озвучка
- Нужно 15-30 минут **чистого голоса без музыки**
- Положить в `data/burunov_raw/`

### Шаг 2. Арендовать GPU (5 минут)
- Запустить сервер с `cloud_init_setup.sh` в User data
- Через 5-7 минут всё установлено автоматически

### Шаг 3. Обучить модели (3-4 часа)
- `scp` аудио на сервер
- `python gpu_train.py`
- Скачать `models.zip`

### Шаг 4. Получить SSH к G1 (когда дадут)
- `scp diag.sh unitree@<G1_IP>:~/`
- `ssh unitree@<G1_IP> "bash diag.sh > diag.txt"`
- Скинуть diag.txt мне — анализирую

### Шаг 5. Деплой на G1 (1-2 часа)
- Установить Ollama + Gemma на G1
- Установить unitree_sdk2_python
- Скопировать models.zip → распаковать
- `docker compose up -d`
- `python robot_controller.py "Штирлиц"` — тест

### Шаг 6. Демо
- HTTP-сервер на G1 (порт 8002)
- Управление с телефона: ввод темы → анекдот
- Запас: 10 предгенерённых wav-ок на популярные темы

## Что получает жюри на демо

1. **Робот G1 стоит, балансирует**
2. **С телефона вводишь тему** ("Штирлиц", "колбаса", "Вовочка")
3. **RGB-лента загорается синим** — робот "думает"
4. **Робот лёгкое покачивание** — "вспоминает"
5. **Голос Бурунова** из динамика:
   > "Штирлиц подошёл к окну. Из окна дуло... Хм, подумал Штирлиц,
   > надо закрыть... Ну, ты понял..."
6. **На панчлайне — резкий жест, кисти делают "большой палец вверх"**
7. **После — робот приседает от смеха**

Полный оффлайн. Никакого облака. Никакого WiFi. AI работает прямо на роботе.
